# Лабораторная работа 2 — Игра «Лови шарик» (JavaFX, MVVM)

## Описание

Проект представляет собой интерактивную игру «Лови шарик», реализованную на JavaFX с архитектурой **Model–View–ViewModel (MVVM)**.  
Игрок кликает по движущемуся шарику, чтобы набирать очки.  
Игра включает бонус‑режим, паузу, таймер, визуальные эффекты и экран завершения с итоговым счётом.

**Индивидуальный вариант №7 — раз в 10 секунд шарик становится полупрозрачным на 3 секунды, очки ×2.**

---

## Возможности игры

- движение шарика с отражением от границ;
- начисление очков при попадании;
- бонус‑режим:
    - каждые 10 секунд активируется на 3 секунды;
    - шарик становится полупрозрачным;
    - цвет меняется плавно;
    - очки ×2;
- пауза по двойному клику;
- отскок шарика от курсора **после клика мимо**;
- таймер обратного отсчёта (60 секунд);
- экран завершения игры:
    - надпись «Игра окончена»;
    - вывод итогового счёта;
    - блокировка игрового поля;
- улучшенный дизайн через CSS:
    - объёмный шарик;
    - мягкие тени;
    - градиентный фон;
    - аккуратная верхняя панель.

## Архитектура проекта (MVVM)

### Model (`GameModel`)
- хранит состояние игры;
- содержит только данные и свойства (`IntegerProperty`, `BooleanProperty`, `DoubleProperty`);
- не содержит JavaFX‑UI.

### View (`GameView.fxml`)
- описывает интерфейс;
- содержит только визуальные элементы;
- не содержит логики.

### ViewModel (`GameViewModel`)
- содержит игровую механику:
    - движение шарика;
    - бонус‑режим;
    - таймер;
    - отскок после клика;
- предоставляет свойства для View;
- не знает о UI‑компонентах.

### Controller (`GameController`)
- связывает View ↔ ViewModel;
- обрабатывает клики пользователя;
- отвечает за визуальные эффекты (прозрачность, цвет, экран завершения);
- не содержит игровой логики.
